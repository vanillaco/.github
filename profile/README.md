# Vanilla & Co.
> The complexity was optional. We opted out.
---
**Considered work. No ornament. No exceptions.**
---
1. Software extends what people can do. Everything else is overhead.
2. A framework that needs a tool that wraps a library that patches a thing that was never broken is not progress.
3. Complexity is a cost, not a feature. Every dependency is a liability.
4. The platform has what you need. Look there first.
5. Write code the next person can read.
6. Add nothing that need not exist. Remove nothing that works.
7. Popularity is not a reason. Neither is novelty.
8. New is not better. Better is better.
9. Ship nothing you cannot explain.
10. Considered software is not a limitation. It is a standard.
---
*Considered software for complicated times. — vanillaco*
